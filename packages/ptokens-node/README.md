# ptokens-node

Module for interacting with a Node.

&nbsp;

***

&nbsp;

### Installation:

```
npm install ptokens-node
```

&nbsp;

***

&nbsp;

### Usage:

```js
import { Node } from 'ptokens-node'

const node = new Node({
  pToken: 'pToken name'
  hostBlockchain: 'ex ETH, EOS',
  endpoint: 'https://...'
})
```
